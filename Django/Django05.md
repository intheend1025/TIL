# Django05

### Authentication system

#### 쿠키

- 서버가 사용자의 웹브라우저에 전송하는 작은 데이터 조각

- 상태가 있는 세션을 만들어줌
- application 에 cookies

- 사용목적
  - 세션관리 (로그인, 공지 안보기, 팝업체크)
  - 개인화 (사용자 선호, 테마)
  - 트래킹 (사용자 행동 기록 분석)

#### 세션

- 상태 유지시키는 것
- 세션 아이디를 쿠키에 저장
- ex) 로그인, 장바구니 등

1. 로그인
2. 로그인성공 (사이트에서 session id를 줌)
3. 다른 곳 클릭할때마다 서버에 session id와 함께 전달

#### Session in Django

- 장고 세션은 미들웨어를 통해 구현
